# 環境構築とプロジェクト作成
## 環境構築
Viteを使うのがおススメ  
-> Node.jsやJSのライブラリやフレームワークでプロジェクトを立ち上げる時に開発環境を整えてくれるビルドツール  
[Vite公式サイト](https://ja.vite.dev/guide/#%E6%9C%80%E5%88%9D%E3%81%AE-vite-%E3%83%95%E3%82%9A%E3%83%AD%E3%82%B7%E3%82%99%E3%82%A7%E3%82%AF%E3%83%88%E3%82%92%E7%94%9F%E6%88%90%E3%81%99%E3%82%8B)

1. 最初の Vite プロジェクトを生成するのコマンドをコピーし、@latestを@6.2.1(動画時点の最新バージョン)に書き換えて、VSCODEのターミナルで実行  
-> 6.2.1とすることで、対話形式でセットアップが開始される  
> npm create vite@6.2.1

3. npm installコマンドを実行する
4. npm run devコマンドを実行して、ローカルサーバーが立ち上がるかチェック  
-> Viteのデフォルトページがローカルサーバーが立ち上がる

Reactのような近年はやりのjsライブラリやフレームワークはNode.jsの仕様を前提としているため、ローカルサーバーを立てて実装を始める  
Viteにはホットリロード機能もあるので、変更が即時反映される
