# ルーティング
ルーティングとはアプリの中でページを切り替える仕組みのこと  
通常はページが遷移してURLが変わり、ページをまるごと書き換える  
ReactではURL変更時にアプリの中身だけを書き換えるから必要な部分だけ切り替えるから早くて便利 
-> **React Router**というライブラリが必要
[React Router](https://reactrouter.com/)  

## React Router導入
1. 以下のコマンドでReact Routerをインストールする  
> npm i react-router-dom  

2. ページを作る: トップページなど
3. App.jsxでルーティングを設定する
  1. **import { BrowserRouter } from 'react-router-dom'で読み込む**
  2. returnの中のタグを<BrowserRouter></BrowserRouter>で囲む
  -> BrowserRouter内でルーティング設定すると、WEBブラウザーにおけるルーティング設定が設定できる  
  3. BrowserRouter内で<Routes>タグを記述し、ルートをグループ化する
  4. Routeタグでルーティング設定する
  -> path属性にURL、element属性に呼び出したいコンポーネントを設定する  
  5. Linkコンポーネントでページを切り替える
  -> 通常ならaタグを使うが、ReactではLinkコンポーネントでページ遷移させる
```jsx
// App.jsx
import './App.css'
import { BrowserRouter, Route, Routes } from 'react-router-dom'
import Home from './pages/Home'
import SamplePage from './pages/SamplePage'

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path='/' element={<Home />}></Route>
        <Route path='/sample-page' element={<SamplePage />}></Route>
      </Routes>
    </BrowserRouter>
  )
}
export default App

// pages/Home.jsx
import { Link } from "react-router-dom";

function Home(){
	return (
		<>
			<h1>Home</h1>
			<Link to='/sample-page' target="_blank">Sample Pageへ!</Link>
		</>
	)
}

export default Home;
```
