# コードの省略
現場レベルでは省略して記述する部分を修正する
## 関数全般
### 引数の()
引数の()は引数が1つだけの場合は省略可能
```jsx
// 修正前
setAnswerLogs((prev) => [...prev, true]);
// 修正後
setAnswerLogs(prev => [...prev, true]);
```

### 関数の処理が1行のみ
関数の処理が1行だけで値をreturnで返すだけなら、{}とreturnは省略可能
```jsx
// 修正前
const correctNum = answerLogs.filter(answer => {
  return answer === true
});
// 修正後
const correctNum = answerLogs.filter(answer => answer === true);
```
